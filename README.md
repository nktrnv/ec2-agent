# EC2 agent

Synchronizes login credentials from the Instance Metadata Service (IMDS).

## Installation

### Debian

```bash
wget https://github.com/C2Devel/ec2-agent/releases/download/v0.1.0/ec2-agent_0.1.0_linux_amd64.deb
sudo apt install ./ec2-agent_0.1.0_linux_amd64.deb
rm ./ec2-agent_0.1.0_linux_amd64.deb
```

### RPM

```bash
wget https://github.com/C2Devel/ec2-agent/releases/download/v0.1.0/ec2-agent_0.1.0_linux_amd64.rpm
sudo yum install ./ec2-agent_0.1.0_linux_amd64.rpm
rm ./ec2-agent_0.1.0_linux_amd64.rpm
```
